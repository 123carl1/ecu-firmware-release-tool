# ECU Firmware Release Tool 0.2.0

这是自动更新链路的引导版本，统一了 GUI、CLI、安装包和在线更新信息的版本身份。

本版本提供图莫斯 USB2XXX 与同星 TSMaster 通道扫描、CAN/LIN 诊断和受项目配置约束的 OTA 操作。安装包通过标签发布流程构建和审计；客户端更新信息使用独立 Ed25519 密钥签名，并校验安装包大小与 SHA-256。

Windows 安装包当前没有软件代码签名证书，系统会显示“未知发布者”。Ed25519 更新签名只用于本工具校验 GitHub 更新资源完整性和来源，不代表 Windows 受信任发布者身份。

USB2XXX 运行库来自固定的官方示例仓库提交；该上游仓库未发现明确再分发许可，具体来源、文件哈希和风险边界见 `THIRD_PARTY_NOTICES.txt`。同星 DLL 不随安装包发布。
